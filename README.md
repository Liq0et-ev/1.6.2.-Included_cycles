# 1.6.2.Included_cycles
# 10x10 divdimensiju masīva aizpildīšana 

Programma izveido divdimensiju masīvu ar 10 rindām un 10 kolonnām un nolasa vienu veselu skaitli no ievades.<br>
Ja ievadītais skaitlis ir 1, masīvs ar iekļautiem cikliem tiek aizpildīts .<br> 
Pēc aizpildīšanas masīvs tiek izvadīts ekrānā tabulas veidā. <br>
Ja ievadītais skaitlis nav 1 vai 2, programma izvada kļūdas paziņojumu.<br>
